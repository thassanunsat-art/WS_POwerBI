# Workshop 1 การนำเข้าและจัดการข้อมูลด้วย Power BI 

## 🚀 ขั้นตอนที่ 1: เปิด Power BI Desktop
**สำหรับผู้เข้าอบรม:**
1. ดับเบิลคลิกที่โปรแกรม **Power BI Desktop**
2. รอจนหน้าจอเริ่มต้น (Start Screen) แสดงขึ้นมา

## 🚀 ขั้นตอนที่ 2: ขั้นตอนการนำเข้าข้อมูล:
1. คลิกที่เมนู **Home**
2. คลิกปุ่ม **Get data**
3. เลือก **Excel**
4. คลิกปุ่ม **Connect**
5. เลือกไฟล์ `workshop_data.xlsx`
6. คลิกปุ่ม **Open**

---

## 🔍 ขั้นตอนที่ 3: เลือกตาราง และเข้า Power Query
**ขั้นตอนการคลิก:**
1. ในหน้าต่าง Navigator ให้ติ๊ก ☑️ เลือกตารางข้อมูลที่ต้องการ
2. คลิกปุ่ม **Transform Data**

> ⚠️ **ข้อควรระวัง:** ✅ **ห้ามกดปุ่ม Load ในขั้นตอนนี้เด็ดขาด**

---

## 💡 ขั้นตอนที่ 4: ทำความรู้จัก Power Query Editor
**ให้ผู้เข้าอบรมสังเกตส่วนประกอบของหน้าจอ:**
* **ด้านซ้าย (Queries):** แสดงรายชื่อตารางที่กำลังทำงานอยู่
* **ตรงกลาง:** แสดงข้อมูลในตาราง
* **ด้านขวา (Applied Steps):** แสดงประวัติและขั้นตอนการจัดการข้อมูลทั้งหมด (สามารถลบหรือย้อนกลับได้)

---

## 🗑️ ขั้นตอนที่ 5: ลบคอลัมน์ที่ไม่จำเป็น
**ขั้นตอนการคลิก:**
1. คลิกเลือกคอลัมน์ที่ไม่ต้องการใช้งาน
2. คลิกขวาที่หัวคอลัมน์
3. เลือก **Remove**
*(ทำซ้ำขั้นตอนเดิมจนครบทุกคอลัมน์ที่ไม่ต้องการ)*

---

## 🔄 ขั้นตอนที่ 6: เปลี่ยนชนิดข้อมูล (Data Type)
**ขั้นตอนการคลิก:**
1. คลิกที่คอลัมน์ **วันที่**
2. ที่แถบเมนูให้เลือก **Data Type**
3. เลือกชนิดข้อมูลเป็น **Date**
4. คลิก **Replace Current** (หากมีป๊อปอัปแจ้งเตือน)

**ทำแบบเดียวกันกับคอลัมน์อื่นๆ:**
* คอลัมน์ **จำนวน** ➡️ เปลี่ยนชนิดข้อมูลเป็น **Whole Number**

---

## ✏️ ขั้นตอนที่ 7: เปลี่ยนชื่อคอลัมน์ให้อ่านง่าย
**ขั้นตอนการคลิก:**
1. ดับเบิลคลิกที่ชื่อหัวคอลัมน์เดิม
2. พิมพ์ชื่อใหม่ที่ต้องการ (เช่น `"จำนวนผู้รับบริการ"`)
3. กด **Enter**

---

## 📥 ขั้นตอนที่ 8: โหลดข้อมูลเข้าสู่ Power BI
**ขั้นตอนการคลิก:**
1. คลิกที่เมนู **Home**
2. คลิกปุ่ม **Close & Apply**
3. ⏳ รอให้ระบบทำการโหลดข้อมูลเข้าสู่หน้าต่างทำงานหลัก
