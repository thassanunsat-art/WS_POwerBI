# 📥 คู่มือการดาวน์โหลดและติดตั้ง Power BI Desktop

การติดตั้ง Power BI Desktop สามารถทำได้ 2 วิธี โดย**ขอแนะนำวิธีที่ 1** เพราะระบบจะทำการอัปเดตเวอร์ชันใหม่ให้คุณโดยอัตโนมัติในทุกๆ เดือนครับ

---

## 🌟 วิธีที่ 1: ติดตั้งผ่าน Microsoft Store (แนะนำ 👍)
**ข้อดี:** โปรแกรมจะอัปเดตตัวเองอัตโนมัติ ไม่ต้องคอยดาวน์โหลดไฟล์ใหม่ทุกเดือน
**ขั้นตอน:**
1. คลิกที่ปุ่ม Start (โลโก้ Windows) บนหน้าจอคอมพิวเตอร์
2. ค้นหาและเปิดแอปพลิเคชัน **Microsoft Store**
3. ที่ช่องค้นหา (Search) ด้านบน ให้พิมพ์คำว่า `Power BI Desktop`
4. คลิกเลือกแอป **Power BI Desktop** (ไอคอนสีเหลือง)
5. คลิกปุ่ม **Get** (รับ) หรือ **Install** (ติดตั้ง)
6. ⏳ รอให้ระบบทำการดาวน์โหลดและติดตั้งจนเสร็จสิ้น จากนั้นคลิก **Open** เพื่อเริ่มใช้งานได้เลย!

---

## 🌐 วิธีที่ 2: ดาวน์โหลดไฟล์ติดตั้ง (.exe) จากเว็บไซต์ Microsoft
**ข้อดี:** เหมาะสำหรับคอมพิวเตอร์ของหน่วยงานที่บล็อกการเข้าถึง Microsoft Store
**ขั้นตอน:**
1. เปิดเว็บเบราว์เซอร์ แล้วเข้าไปที่ลิงก์: [https://powerbi.microsoft.com/desktop/](https://powerbi.microsoft.com/desktop/)
2. หากต้องการเลือกภาษาในการติดตั้ง ให้คลิกที่ **See download or language options** (ดูตัวเลือกการดาวน์โหลดหรือภาษา)
3. เลือกภาษาเป็น **English** (แนะนำให้ใช้เมนูภาษาอังกฤษเพื่อให้ตรงกับคู่มือการสอนส่วนใหญ่)
4. คลิกปุ่ม **Download**
5. ติ๊กเลือกไฟล์ติดตั้งที่ตรงกับระบบของคุณ (ส่วนใหญ่ปัจจุบันใช้แบบ 64-bit ให้เลือก `PBIDesktopSetup_x64.exe`) แล้วคลิก **Next**
6. เมื่อดาวน์โหลดเสร็จสิ้น ให้ **ดับเบิลคลิกไฟล์ที่โหลดมา** เพื่อเริ่มการติดตั้ง
7. กดปุ่ม **Next** ➡️ กดยอมรับเงื่อนไข (Accept the terms) ➡️ กด **Next** ไปเรื่อยๆ จนถึงปุ่ม **Install**
8. ⏳ รอระบบติดตั้งจนเสร็จ แล้วกดปุ่ม **Finish**

---

## ⚙️ สเปคคอมพิวเตอร์ขั้นต่ำที่แนะนำ (System Requirements)
* **ระบบปฏิบัติการ:** Windows 10 หรือ Windows 11
* **RAM:** ขั้นต่ำ 4 GB (แนะนำให้ใช้ 8 GB ขึ้นไป เพื่อการประมวลผลที่ลื่นไหล)
* **หน้าจอ:** ความละเอียดอย่างน้อย 1440x900 หรือ 1600x900 (16:9)
