# Get Data
# 📊 คู่มือขั้นตอนการสร้าง Dashboard บน Power BI (ฉบับมือใหม่)

การสร้าง Dashboard ใน Power BI ให้ประสบความสำเร็จและตอบโจทย์ธุรกิจ มีขั้นตอนหลักๆ 5 ขั้นตอน ดังนี้ครับ:

---

## 🛠️ ขั้นตอนที่ 1: การเตรียมและนำเข้าข้อมูล (Data Preparation & Connect)
เริ่มต้นด้วยการนำข้อมูลจากแหล่งต่างๆ เข้าสู่ **Power BI Desktop**

1. เปิดโปรแกรม **Power BI Desktop**
2. ไปที่แถบเมนู `Home` -> คลิก **Get Data**
3. เลือกแหล่งข้อมูลที่ต้องการ (เช่น Excel, SQL Server, Web หรือ CSV)
4. คลิก **Transform Data** เพื่อเข้าสู่ *Power Query Editor* เพื่อทำความสะอาดข้อมูล (Data Cleaning) เช่น:
   * ลบแถวที่ว่างหรือซ้ำซ้อน
   * เปลี่ยนประเภทข้อมูล (Data Type) ให้ถูกต้อง เช่น ตัวเลข, วันที่, ข้อความ
   * แยกคอลัมน์ หรือรวมคอลัมน์ตามต้องการ
5. เมื่อเสร็จแล้วให้คลิก **Close & Apply**

---

## 📐 ขั้นตอนที่ 2: การสร้างความสัมพันธ์และคำนวณข้อมูล (Data Modeling & DAX)
หากมีข้อมูลหลายตาราง จำเป็นต้องนำมาเชื่อมโยงกันเพื่อให้รายงานแสดงผลได้อย่างถูกต้อง

1. ไปที่มุมมอง **Model view** (ไอคอนรูปตารางเชื่อมกันด้านซ้ายมือ)
2. ลากเส้นเชื่อมโยงคอลัมน์ที่เหมือนกันระหว่างตาราง (เช่น `ProductID` จากตารางยอดขาย ไปยัง `ProductID` จากตารางสินค้า) เพื่อสร้าง **Relationship**
3. กลับมาที่มุมมอง **Report view** เพื่อเขียนสูตรคำนวณเพิ่มเติมด้วย **DAX (Data Analysis Expressions)** (ถ้ามี) เช่น:
   * สร้าง **New Measure** เพื่อหาผลรวมยอดขาย:
     ```dax
     Total Sales = SUM(Sales[Revenue])
     ```

---

## 🎨 ขั้นตอนที่ 3: การสร้างรายงานและกราฟ (Creating Report/Visualizations)
ขั้นตอนนี้คือการนำข้อมูลมาเปลี่ยนเป็นภาพหรือกราฟที่เข้าใจง่าย

1. ที่แถบ **Visualizations** ด้านขวามือ ให้เลือกรูปแบบกราฟที่เหมาะสม เช่น:
   * **Bar / Column Chart:** สำหรับเปรียบเทียบข้อมูล
   * **Line Chart:** สำหรับดูแนวโน้ม (Trend) ตามเวลา
   * **Card:** สำหรับแสดงตัวเลขสำคัญตัวเดียว (เช่น ยอดขายรวม)
   * **Slicer:** สำหรับทำตัวกรองข้อมูล (เช่น ตัวเลือกปี, ตัวเลือกภูมิภาค)
2. ลากคอลัมน์ข้อมูลจากแถบ **Data** มาใส่ในช่อง `X-axis`, `Y-axis` หรือ `Legend` ของกราฟนั้นๆ
3. ตกแต่งกราฟที่ไอคอน **Format visual** (รูปแปรงทาสี) เพื่อเปลี่ยนสี, ใส่ชื่อกราฟ, หรือเปิดเปิด Data Labels ให้เปิดอ่านง่าย

---

## 🚀 ขั้นตอนที่ 4: การเผยแพร่รายงาน (Publish Report)
เมื่อออกแบบรายงานใน Power BI Desktop เสร็จแล้ว ต้องนำขึ้นระบบ Cloud เพื่อสร้างเป็น Dashboard

1. กดปุ่ม **Save** เซฟไฟล์งานของคุณก่อน (`.pbix`)
2. ที่แถบ `Home` -> คลิกปุ่ม **Publish** (อยู่ด้านขวาสุด)
3. ลงชื่อเข้าใช้งานด้วยบัญชี Power BI Work/School
4. เลือก **Workspace** ที่ต้องการนำรายงานไปใส่ (หากทดสอบให้เลือก *My Workspace*) แล้วกด **Select**
5. รอจนระบบขึ้นว่า *Success*

---

## 📌 ขั้นตอนที่ 5: การปักหมุดสร้าง Dashboard บน Power BI Service
*หมายเหตุ: ในความหมายของ Power BI คำว่า "Dashboard" จะสร้างขึ้นบน **Power BI Service (Cloud)** โดยการรวมกราฟเด่นๆ จากหลายรายงานมาไว้ในหน้าเดียว*

1. เปิดเว็บเบราว์เซอร์ไปที่ [Power BI Service](https://app.powerbi.com) และเข้าสู่ระบบ
2. ไปที่ **Workspace** ที่คุณเพิ่งทำการ Publish รายงานขึ้นมา
3. คลิกเปิด **Report** ที่คุณสร้างไว้
4. เอาเมาส์ไปชี้ที่กราฟที่ต้องการ จะปรากฏไอคอนรูป **หมุด (Pin visual)** ขึ้นมา ให้คลิกที่ไอคอนนั้น
5. จะมีหน้าต่างเด้งขึ้นมา ให้เลือก:
   * **New dashboard:** ตั้งชื่อ Dashboard ใหม่ของคุณ
   * **Existing dashboard:** เลือกปักหมุดใน Dashboard ที่มีอยู่แล้ว
6. คลิก **Pin** ทำซ้ำแบบนี้กับกราฟอื่นๆ ที่สำคัญ จนได้หน้า Dashboard ที่สรุปภาพรวมทั้งหมดตามต้องการ
