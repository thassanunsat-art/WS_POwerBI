# Workshop 1 การนำเข้าและจัดการข้อมูลด้วย Power BI 

## 🚀 ขั้นตอนที่ 1: เปิด Power BI Desktop
**สำหรับผู้เข้าอบรม:**
1. ดับเบิลคลิกที่โปรแกรม **Power BI Desktop**
2. รอจนหน้าจอเริ่มต้น (Start Screen) แสดงขึ้นมา

## 🚀 ขั้นตอนที่ 2: ขั้นตอนการนำเข้าข้อมูล:
1. คลิกที่เมนู **Home**
2. คลิกปุ่ม **Get data**
3. เลือก **Excel**
4. คลิกปุ่ม **Connect**
5. เลือกไฟล์ `workshop_all_provinces.xlsx`
6. คลิกปุ่ม **Open**

---

## 🔍 ขั้นตอนที่ 3: เลือกตาราง และเข้า Power Query
**ขั้นตอนการคลิก:**
1. ในหน้าต่าง Navigator ให้ติ๊ก ☑️ เลือกตารางข้อมูลที่ต้องการ
2. คลิกปุ่ม **Transform Data**

## 💡 ขั้นตอนที่ 4: ทำความรู้จัก Power Query Editor
**ให้ผู้เข้าอบรมสังเกตส่วนประกอบของหน้าจอ:**
* **ด้านซ้าย (Queries):** แสดงรายชื่อตารางที่กำลังทำงานอยู่
* **ตรงกลาง:** แสดงข้อมูลในตาราง
* **ด้านขวา (Applied Steps):** แสดงประวัติและขั้นตอนการจัดการข้อมูลทั้งหมด (สามารถลบหรือย้อนกลับได้)
