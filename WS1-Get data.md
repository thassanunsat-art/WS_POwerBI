# Workshop 1 การนำเข้าและจัดการข้อมูลด้วย Power BI 

## 🚀 ขั้นตอนที่ 1: เปิด Power BI Desktop
**สำหรับผู้เข้าอบรม:**
1. ดับเบิลคลิกที่โปรแกรม **Power BI Desktop**
2. รอจนหน้าจอเริ่มต้น (Start Screen) แสดงขึ้นมา

## 🚀 ขั้นตอนที่ 2: ขั้นตอนการนำเข้าข้อมูล:
 ### 💻 Excel,CSV
1. คลิกที่เมนู **Home**
2. คลิกปุ่ม **Get data**
3. เลือก **Excel**
4. คลิกปุ่ม **Connect**
5. เลือกไฟล์ `workshop_all_provinces_dirty.xlsx`
6. คลิกปุ่ม **Open**

   <img width="40%" height="40%" alt="image" src="https://github.com/user-attachments/assets/7730cfc7-2ac7-4479-852c-f6d25551f430" />


### 💻 SRHRE POINT 
1. คลิกที่เมนู **Home**
2. คลิกปุ่ม **Get data**
3. เลือก **More-SharePoint folder**
4. คลิกปุ่ม **Connect**
5. Site URL `(https://nhcothai.sharepoint.com/sites/Project_Data_visualization)` ลิงก์ sharepoint ที่เก็บข้อมูล
6. คลิกปุ่ม **Ok**

**ขั้นตอนการคลิก:**

## 💡 ขั้นตอนที่ 4: ทำความรู้จัก Power Query Editor
**ให้ผู้เข้าอบรมสังเกตส่วนประกอบของหน้าจอ:**
* **ด้านซ้าย (Queries):** แสดงรายชื่อตารางที่กำลังทำงานอยู่
* **ตรงกลาง:** แสดงข้อมูลในตาราง
* **ด้านขวา (Applied Steps):** แสดงประวัติและขั้นตอนการจัดการข้อมูลทั้งหมด (สามารถลบหรือย้อนกลับได้)
