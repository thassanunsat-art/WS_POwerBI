# 📉 สร้าง Line Chart, Card, แผนที่ และการจัดหน้า Report

## 🔹 Step 1: สร้าง Line Chart ดูแนวโน้ม (Trend Analysis)
**ขั้นตอน:**
1. คลิกพื้นที่ว่างบนหน้าจอ (Canvas)
2. ไปที่แผง Visualizations ➡️ คลิกไอคอน **Line Chart** (กราฟเส้น)
3. การตั้งค่าข้อมูล (Data):
   * **X-axis (แกน X):** ลากคอลัมน์ `วันที่` หรือ `เดือน` มาวาง
   * **Y-axis (แกน Y):** ลากคอลัมน์ `จำนวนผู้รับบริการ` มาวาง
4. **ปรับแต่งให้สวยงาม:** ไปที่ไอคอนแปรงทาสี ➡️ เลือก **Lines** ➡️ ปรับ **Stroke width** ให้เส้นหนาขึ้น และเปลี่ยน **Line type** เป็น **Smooth** เพื่อให้เส้นกราฟโค้งมนดูสมูท

**💡**
> *"กราฟเส้น (Line Chart) มีวัตถุประสงค์เดียวเลยคือเพื่อดู 'แนวโน้มตามช่วงเวลา' (Trend over Time) ช่วยให้ผู้บริหารมองออกว่าช่วงไหนผู้รับบริการพีคสุด หรือช่วงไหนที่ยอดดรอปลง"*

---

## 🔹 Step 2: เพิ่ม Card แสดงค่ารวม (Big Bold Number)
**ขั้นตอน:**
1. คลิกพื้นที่ว่างบนหน้าจอ
2. ไปที่แผง Visualizations ➡️ คลิกไอคอน **Card** (รูปกล่องที่มีตัวเลข `123`)
3. ลากคอลัมน์ `จำนวนผู้รับบริการ` มาวางที่ช่อง **Fields** (ระบบจะคำนวณเป็นยอดรวม Sum ให้อัตโนมัติ)
4. **ปรับแต่งตัวเลข:** ไปที่ไอคอนแปรงทาสี ➡️ **Callout value** ➡️ ปรับขนาดฟอนต์ให้ใหญ่เด่นชัด (เช่น 35-40 pt) และเปลี่ยนสีฟอนต์ตามต้องการ

**💡 **
> *"Card Visual คือสิ่งแรกที่ผู้บริหารจะมองหาบน Dashboard มันคือการสรุปยอดรวมทั้งหมด (KPI) ให้เห็นเป็นตัวเลขตัวใหญ่ๆ ชัดๆ แบบไม่ต้องเสียเวลากวาดสายตาหา"*

---

## 🔹 Step 3: สร้างแผนที่ (Map Visual)
**ขั้นตอน:**
1. คลิกพื้นที่ว่างบนหน้าจอ
2. ไปที่แผง Visualizations ➡️ คลิกไอคอน **Map** (รูปโลก) หรือ **Filled Map**
3. การตั้งค่าข้อมูล (Data):
   * **Location:** ลากคอลัมน์ `จังหวัด` (จากตาราง Lookup) มาวาง
   * **Bubble size:** ลากคอลัมน์ `จำนวนผู้รับบริการ` มาวาง (เพื่อให้จังหวัดที่มียอดเยอะ วงกลมจะใหญ่ตาม)
4. *⚠️ หมายเหตุ : หากแผนที่ขึ้นเตือนให้เปิดสิทธิ์ ให้ไปที่ File -> Options and settings -> Options -> Security -> ติ๊กถูกที่ "Use Map and Filled Map visuals"*

**💡 **
> *"เมื่อไหร่ก็ตามที่ข้อมูลของเรามีมิติของพื้นที่ (Geo-spatial) เช่น จังหวัด ภูมิภาค การใช้ Map จะช่วยดึงดูดสายตาและทำให้เห็นการกระจายตัวของข้อมูลเชิงพื้นที่ได้ดีที่สุด"*

---

## 🔹 Step 4: เทคนิคการจัดหน้า Report ให้ดูง่าย (Dashboard Layout)
**ขั้นตอนการจัดวาง :**
1. **ส่วนบนสุด (Top Banner):** วางชื่อเรื่องรายงาน (Text Box) ไว้มุมซ้ายบนสุด
2. **ฝั่งซ้ายหรือแถวบน (Filter Area):** นำ **Slicer** ทั้งหมด (เขตสุขภาพ, เดือน) ไปเรียงไว้แถวบนสุดหรือแถวซ้ายสุด เพื่อให้คนใช้งานเห็นและกดเลือกก่อน
3. **ส่วนข้อมูลสำคัญ (KPI Area):** นำกล่อง **Card** ยอดรวม ไปวางไว้ถัดจาก Slicer หรือมุมบนขวา เพื่อให้เห็นตัวเลขภาพรวมทันที
4. **ส่วนวิเคราะห์ (Detail Area):** นำ **Bar Chart** และ **Line Chart / Map** วางตรงกลางและด้านล่าง โดยขยายขนาดให้ใหญ่เห็นรายละเอียดชัดเจน
5. **การจัดแนว:** คลิกเลือก Visual ➡️ ไปที่เมนู **Format** ด้านบน ➡️ ใช้ปุ่ม **Align** เพื่อจัดให้ขอบกราฟตรงกันอย่างเป็นระเบียบ

**💡 **
> *"การจัดหน้า Dashboard ที่ดีต้องไหลตามสายตาคนอ่านจาก 'บนลงล่าง' และ 'ซ้ายไปขวา' (Z-Pattern) เริ่มจากกรองข้อมูล ➡️ ดูสรุปยอด ➡️ แล้วค่อยเจาะลึกที่ตัวกราฟ"*

---

## 🔹 Step 6: บันทึกไฟล์ (Save & Share)
**ขั้นตอน:**
1. ไปที่เมนู **File** มุมซ้ายบน
2. คลิก **Save As**
3. ตั้งชื่อไฟล์ (เช่น `Workshop`)
4. สังเกตว่าไฟล์ที่ได้จะมีนามสกุลเป็น `.pbix` (ซึ่งเป็นไฟล์โปรเจกต์หลักของ Power BI)
5. คลิกปุ่ม **Save** เป็นอันเสร็จสิ้นเวิร์กชอป!
