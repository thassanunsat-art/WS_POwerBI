# 🛠️ Workshop: การใช้ Power Query

### 📌 เคสที่ 1: วันที่หลายรูปแบบ (Date ไม่สะอาด)
* **ลักษณะปัญหา:** วันที่ในคอลัมน์เดียวกันผสมกัน บางแถวเป็น `dd/mm/yyyy` บางแถว `yyyy-mm-dd` บางแถวเป็น Date จริง
* **💡 ฟีเจอร์:** Change Data Type / Replace Errors / แปลง Text -> Date
* **👇 ขั้นตอนการคลิกแก้:**
  1. คลิกที่ไอคอนชนิดข้อมูล (มุมซ้ายบนของหัวคอลัมน์วันที่)
  2. เลือกเปลี่ยนเป็น **Date** (Change Data Type)
  3. สังเกตแถวที่ฟอร์แมตผิดจะขึ้นคำว่า `Error`
  4. เคลียร์ Error โดยคลิกขวาที่หัวคอลัมน์ -> เลือก **Replace Errors...** แล้วใส่ค่าที่ต้องการ
  5. *(เทคนิคเพิ่มเติม)* หากต้องการแปลงเจาะจง: คลิกขวาหัวคอลัมน์ -> **Change Type** -> **Using Locale...** เลือก Date ให้ตรงกับประเทศต้นทาง

---

### 📌 เคสที่ 2: คอลัมน์เดือนปนกัน (ตัวเลข + ข้อความ)
* **ลักษณะปัญหา:** บางแถวเป็นเลข 1, 2, 3 บางแถวเป็นข้อความ `"1"`, `"2"` 
* **💡 ฟีเจอร์:** Change Type / Transform -> Format / ตรวจสอบ Data Type
* **👇 ขั้นตอนการคลิกแก้:**
  1. คลิกเลือกคอลัมน์เดือน
  2. ไปที่แถบเมนู **Transform** ด้านบน
  3. หมวด Text Column คลิก **Format** -> เลือก **Trim** (ตัดช่องว่างที่มองไม่เห็น)
  4. หากมีเครื่องหมายคำพูด (") ให้คลิกขวาหัวคอลัมน์ -> **Replace Values...**
     * *Value To Find:* พิมพ์ `"`
     * *Replace With:* ปล่อยว่างไว้ แล้วกด OK
  5. เปลี่ยนชนิดข้อมูลที่หัวคอลัมน์ให้เป็น **Whole Number** ให้สม่ำเสมอกัน

---

### 📌 เคสที่ 3: ค่าว่าง (Missing / Null)
* **ลักษณะปัญหา:** หน่วยงานบางแถวว่าง จำนวนผู้รับบริการบางแถวเป็นค่าว่าง (null)
* **💡 ฟีเจอร์:** Replace Values / Remove Blank Rows / Fill Down หรือ Fill Up
* **👇 ขั้นตอนการคลิกแก้:**
  * **กรณีที่ 1: เติมข้อมูลจากแถวบน (Fill Down)**
    1. คลิกขวาที่หัวคอลัมน์หน่วยงาน
    2. เลือก **Fill** -> คลิก **Down**
  * **กรณีที่ 2: เปลี่ยนค่าว่างเป็นเลข 0**
    1. คลิกขวาที่หัวคอลัมน์จำนวนผู้รับบริการ
    2. เลือก **Replace Values...**
    3. *Value To Find:* พิมพ์ `null` | *Replace With:* พิมพ์ `0` -> กด OK
  * **กรณีที่ 3: ลบแถวที่ว่างทิ้งทั้งหมด**
    1. ไปที่เมนู **Home** 2. คลิกปุ่ม **Remove Rows** -> เลือก **Remove Blank Rows**

---

### 📌 เคสที่ 4: ข้อความไม่สม่ำเสมอ (Inconsistent Values)
* **ลักษณะปัญหา:** มีการพิมพ์ทั้งคำเต็ม "ผู้ป่วยนอก" และคำย่อ "ผปน."
* **💡 ฟีเจอร์:** Replace Values / Standardize ข้อมูลเตรียมทำ Dimension
* **👇 ขั้นตอนการคลิกแก้:**
  1. คลิกขวาที่หัวคอลัมน์นั้น
  2. เลือก **Replace Values...**
  3. ในช่อง *Value To Find* พิมพ์ `ผปน.`
  4. ในช่อง *Replace With* พิมพ์ `ผู้ป่วยนอก` -> กด OK

---

### 📌 เคสที่ 5: คอลัมน์เกินความจำเป็น
* **ลักษณะปัญหา:** คอลัมน์ "หมายเหตุ" มีข้อมูลบ้าง ว่างบ้าง ไม่จำเป็นต่อการทำ Report
* **💡 ฟีเจอร์:** Remove Columns / เลือกเก็บเฉพาะข้อมูลที่จำเป็น
* **👇 ขั้นตอนการคลิกแก้:**
  * **วิธีที่ 1 (ลบทิ้ง):** คลิกขวาที่หัวคอลัมน์หมายเหตุ -> เลือก **Remove**
  * **วิธีที่ 2 (เลือกเก็บ - แนะนำ!):** กด `Ctrl` ค้างไว้ -> คลิกเลือกเฉพาะคอลัมน์ที่จำเป็น -> คลิกขวาที่หัวคอลัมน์ใดก็ได้ที่เลือกไว้ -> เลือก **Remove Other Columns**
