# 🧮 การสร้างสูตร DAX (Data Analysis Expressions)

# 📊 Visual ที่แนะนำให้ใช้คู่กัน: * นำไปใส่ในตาราง (Table) ร่วมกับรายชื่อจังหวัด
- ใช้ทำ Conditional Formatting (คลิกขวาที่ชื่อ Measure ในแผง Visual -> เลือก Conditional formatting -> Background color) เพื่อสั่งให้ระบบไฮไลท์สีเข้มจัดที่แถวจังหวัดที่มียอดบริการพุ่งสูงที่สุดโดยอัตโนมัติ

## 📈 สรุป Visual ที่สร้างในหน้า Dashboard 
เพื่อให้ผู้เรียนเห็นการทำงานร่วมกันของทั้ง 5 Parts ให้พาสร้างหน้าจอรายงาน (Report View) โดยลากข้อมูลลง Visual ดังนี้ครับ:
- Card (การแสดงเลข KPI หลัก): สร้าง 2 กล่อง วางเคียงข้างกัน
- กล่องที่ 1 ใส่ [Total Patients]
- กล่องที่ 2 ใส่ [OPD Ratio (%)]
- Bar Chart (เปรียบเทียบเชิงปริมาณ): แกน Y ใส่ "จังหวัด", แกน X ใส่ [Total Patients]
- Stacked Column Chart (เปรียบเทียบสัดส่วน): แกน X ใส่ "ภาค", แกน Y ใส่ [OPD Patients] และ [IPD Patients] ซ้อนกัน
- Slicer (ปุ่มควบคุมดึงดูดสายตา): สร้าง 3 ตัวกรอง วางไว้แถวบนสุด ได้แก่ "ปี", "เดือน" และ "เขตสุขภาพ" เพื่อให้ผู้เรียนลองกดเล่นแล้วดูสูตร DAX คำนวณข้ามมิติตามฟิลเตอร์ทันที

## 🧪 Workshop Part 1: Measure แรก (พื้นฐานที่สุด)
เริ่มต้นด้วยการสร้างยอดรวมพื้นฐาน ซึ่งจะเป็นฐานสำคัญสำหรับนำไปใช้คำนวณใน Measure อื่นๆ ต่อไป

**1️⃣ จำนวนผู้รับบริการทั้งหมด**
```dax
Total Patients = 
SUM ( 'Sheet1'[จำนวนผู้รับบริการ] )
```

📊 *Visual ที่แนะนำให้ใช้คู่กัน: * Card: แสดงตัวเลขรวมเดี่ยวๆ ขนาดใหญ่*

- Table / Matrix: ดึงข้อความ "จังหวัด" มาวางคู่กัน ระบบจะกระจายยอดรวมแยกรายจังหวัดให้โดยอัตโนมัติ
-  Bar Chart: แสดงอันดับผู้รับบริการรายภาค

##  🧪 Workshop Part 2: Measure แยกตามประเภทบริการ

2️⃣ ผู้ป่วยนอกทั้งหมด (OPD)

```dax
OPD Patients = 
CALCULATE (
    [Total Patients],
    'Sheet1'[ประเภทบริการ] = "ผู้ป่วยนอก"
)
```
3️⃣ ผู้ป่วยในทั้งหมด (IPD)

```dax
IPD Patients = 
CALCULATE (
    [Total Patients],
    'Sheet1'[ประเภทบริการ] = "ผู้ป่วยใน"
)
```
📌 จุดสอนสำคัญ (Instructor Notes):
- เน้นย้ำโครงสร้างของฟังก์ชัน CALCULATE( [ชื่อ Measure], เงื่อนไขที่ 1, เงื่อนไขที่ 2, ... )
- อธิบายให้ผู้เรียนเข้าใจว่า CALCULATE ทำหน้าที่เสมือนผู้คุมประตูกรองข้อมูล โดยจะเปลี่ยนบริบทการคำนวณ (Filter Context) ให้เหลือเฉพาะข้อมูลที่ตรงตามเงื่อนไขในวงเล็บเท่านั้น

## 🧪 Workshop Part 3: Measure เชิงเปรียบเทียบ
เรียนรู้การนำฐานข้อมูลจาก Measure ที่สร้างไว้แล้วก่อนหน้า มาคำนวณสัดส่วนเป็นเปอร์เซ็นต์เพื่อเปรียบเทียบข้อมูลอย่างเป็นธรรม

4️⃣ สัดส่วนผู้ป่วยนอก (%)
```dax
OPD Ratio (%) = 
DIVIDE ( [OPD Patients], [Total Patients] )
```
📌 จุดสอนสำคัญ (Instructor Notes):
- ทำไมต้องใช้ DIVIDE แทนเครื่องหมายหาร ( / ) ? อธิบายว่าฟังก์ชัน DIVIDE มีระบบความปลอดภัย (Safe Divide) เมื่อเจอเหตุการณ์ที่ตัวหารเป็นศูนย์ (Divide by Zero) เช่น ในวันนั้นไม่มีผู้รับบริการเลย ระบบจะไม่พังและไม่ขึ้นคำว่า Error แต่จะแสดงผลเป็นค่าว่าง (Blank) หรือใส่ค่าที่เรากำหนดไว้แทน
- เทคนิคการปรับฟอร์แมต: เมื่อเขียนสูตรเสร็จแล้ว ให้คลิกที่ชื่อ Measure นี้ในแถบขวามือ จากนั้นมองขึ้นไปที่เมนู Measure tools ด้านบน แล้วกดเลือกสัญลักษณ์เปอร์เซ็นต์ ( % ) เพื่อเปลี่ยนการแสดงผลจากทศนิยมให้เป็นร้อยละที่อ่านง่าย

## 🧪 Workshop Part 4: Measure ตามช่วงเวลา
ก้าวเข้าสู่การคำนวณขั้นกลาง โดยสรุปข้อมูลในมิติของเวลา และหาค่าเฉลี่ยแบบวนลูปคำนวณ (X-Functions)

5️⃣ จำนวนผู้รับบริการเฉพาะปี 2024
```dax
Patients 2024 = 
CALCULATE (
    [Total Patients],
    'Sheet1'[ปี] = 2024
)
```
6️⃣ ค่าเฉลี่ยผู้รับบริการต่อเดือน

```dax
Average Patients per Month = 
AVERAGEX (
    VALUES ( 'Sheet1'[เดือน] ),
    [Total Patients]
)
```
📌 จุดสอนสำคัญ (Instructor Notes):

- VALUES: ทำหน้าที่ดึงรายชื่อเดือนทั้งหมดในตารางออกมารายการละ 1 คำ (ตัดตัวซ้ำออก) เพื่อสร้างเป็นตารางจำลองในหน่วยความจำ
- AVERAGEX: ระบบจะหยิบตารางจากข้อแรกมา วนลูปคำนวณหายอด [Total Patients] ทีละเดือนจนครบ จากนั้นจึงนำผลลัพธ์ของทุกเดือนมาหาค่าเฉลี่ยร่วมกันในตอนท้าย (เรียกว่ากระบวนการ Context Transition)

## 🧪 Workshop Part 5: Measure สำหรับผู้บริหาร
ตอบโจทย์การทำรายงานวิเคราะห์ขั้นสูงเพื่อส่งต่อให้ผู้บริหารใช้ตัดสินใจ โดยหาค่าสถิติขั้นสูงในระดับรายบุคคลหรือรายพื้นที่

7️⃣ จังหวัดที่มีผู้รับบริการสูงสุด

```dax
Max Patients by Province = 
MAXX (
    VALUES ( 'Sheet1'[จังหวัด] ),
    [Total Patients]
)
```
